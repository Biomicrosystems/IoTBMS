# Dispositivos

Los dispositivos son el componente principal de la aplicación. Piensa en los dispositivos como cada microcontrolador que se conecta. Un dispositivo puede tener diferentes funciones, y la forma en que la aplicación se comunica con el microcontrolador es mediante el protocolo UART. Por esta razón, el dispositivo debe estar conectado por cable o por Bluetooth.

Esto hace que esta aplicación sea perfecta para dispositivos que no requieran ser movilizados continuamente. (¡Estamos trabajando para implementar una conexión completamente inalámbrica!)

Una vez tu dispositivo está conectado, los usuarios podrán enviar comandos desde internet. Esto funciona porque gestionamos los comandos que se crean en la nube y los redirigimos a los dispositivos correctos. Así, no tienes que cambiar el código de tu dispositivo ni lidiar con servidores. Simplemente puedes recibir y enviar mediante comandos de puerto serial. Tú solo te preocupas por el sistema que desarrollas, y nosotros nos encargamos de hacerlo accesible a través de internet.
